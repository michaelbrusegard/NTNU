What is specificity in CSS, no need to write more than 4-5 sentences
Specificity is the way that the browser decides which CSS rule to apply to an element. 
The browser will apply the rule with the highest specificity. 
The specificity is determined by the number of each type of selector in the rule. 
The order of specificity is as follows: 1. Inline styles 2. IDs 3. Classes, attributes and pseudo-classes 4. Elements and pseudo-elements
